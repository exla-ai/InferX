# Hardware & Model Compatibility

## Supported Hardware Platforms

### NVIDIA GPUs
- NVIDIA RTX 3090
- NVIDIA RTX 4090
- NVIDIA Jetson Series
  - Jetson AGX Orin
  - Jetson Orin Nano
  - Jetson AGX Xavier
  - Jetson Nano
  - Jetson TX2

### Edge Devices
- Raspberry Pi 4
- Intel NUC
- Various ARM-based devices

## Supported Models

### Language Models
- DeepSeek
  - Supported operations: Text generation, embeddings
  - Optimized for: RTX 3090, RTX 4090
  - Quantization: 4-bit, 8-bit options available

### Vision Models
- CLIP (Contrastive Language-Image Pretraining)
  - Supported operations: Image-text similarity, zero-shot classification
  - Optimized for: Jetson AGX Orin, Jetson Orin Nano, RTX 3090, RTX 4090
  - Special optimizations: NVIDIA PyTorch wheel for Jetson devices
  - Models: openai/clip-vit-large-patch14-336 (default for Jetson), openai/clip-vit-base-patch32 (default for CPU)

- MobileNet (Coming soon)
- ResNet34 (Coming soon)

## Python Version Compatibility

| Model | Python 3.8 | Python 3.9 | Python 3.10 | Python 3.11 |
|-------|------------|------------|-------------|-------------|
| DeepSeek | ✅ | ✅ | ✅ | ✅ |
| CLIP (CPU) | ✅ | ✅ | ✅ | ✅ |
| CLIP (Jetson GPU) | ❌ | ❌ | ✅ | ❌ |
| CLIP (Standard GPU) | ✅ | ✅ | ✅ | ✅ |

**Note**: For Jetson devices, Python 3.10 is required for GPU acceleration with the NVIDIA PyTorch wheel. Other Python versions will fall back to CPU execution.
