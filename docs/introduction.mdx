# Welcome to Exla

Exla is a powerful inference engine designed to run Large Language Models (LLMs) directly in Elixir. It provides high-performance capabilities while maintaining the elegance and reliability of the Elixir ecosystem.

## Key Features

- **Native Elixir Implementation**: Built from the ground up for the BEAM VM
- **High Performance**: Optimized for efficient model inference
- **Multiple Model Support**: Run various LLM architectures
- **GPU Acceleration**: Leverage GPU capabilities for faster processing
- **Easy Integration**: Simple API for seamless integration into Elixir applications
